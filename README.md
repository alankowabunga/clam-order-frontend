# 海的女兒 - 蛤蠣訂單管理系統前端 (Clam Order Frontend)

這是一個為「海的女兒 蛤蠣專賣」開發的新鮮蛤蠣訂購管理系統前端專案。本系統旨在提供直覺的訂購流程與便利的後台管理功能。

## 🌟 核心功能

### 1. 顧客訂購流程
- **品項瀏覽**：查看今日供應的新鮮蛤蠣品項、規格與價格。
- **多步訂購表單**：
  - **選擇品項**：即時試算總斤數與金額。
  - **配送方式**：支援「台中配送」與「門市自取」。
  - **詳細資訊**：填寫收件人資料、匯款資訊及特殊要求（如管理室代收）。
- **優惠邏輯**：
  - **量販優惠**：訂購滿 10 斤或 20 斤自動扣抵金額。
  - **自取優惠**：選擇門市自取可享有每斤折扣。
  - **運費計算**：依據地區與訂購重量自動判斷是否免運。

### 2. 管理員控制台
- **數據概覽**：今日訂單量、總銷售斤數及待處理案件統計。
- **訂單管理**：查看訂單詳細資料、付款狀態、地區與所需斤數。
- **資料匯出**：支援匯出出貨清單供物流作業使用。

## 🛠 關鍵技術服務

- **前端框架**：[Vue 3](https://vuejs.org/) (Composition API)
- **程式語言**：[TypeScript](https://www.typescriptlang.org/)
- **狀態管理**：[Pinia](https://pinia.vuejs.org/)
- **路由導航**：[Vue Router](https://router.vuejs.org/)
- **圖標庫**：[Lucide Vue Next](https://lucide.dev/guide/packages/lucide-vue-next)
- **HTTP 用戶端**：[Axios](https://axios-http.com/)
- **樣式設計**：Vanilla CSS (具備 Glassmorphism 磨砂玻璃效果與響應式設計)

## 📁 專案架構

```text
src/
├── api/            # API 服務層 (包含 Mock 數據邏輯)
├── assets/         # 靜態資源與樣式
├── domain/         # TypeScript 介面與模型定義
├── router/         # 路由配置
├── store/          # Pinia 狀態管理 (訂單邏輯中心)
├── views/          # 頁面組件 (首頁、訂購、管理)
├── App.vue         # 應用程式入口組件
└── main.ts         # 應用程式初始化
```

## 🚀 業務邏輯摘要

此前端專案包含完整的商業邏輯 Mock 實作（詳見 `src/api/index.ts`）：
- **價格計算**：
  - 大蛤蠣: $170/斤
  - 特大蛤蠣: $200/斤
  - 冬季限定款: $220/斤
- **折扣規則**：
  - 滿 10 斤：每斤折抵 $5
  - 滿 20 斤：每斤折抵 $10
  - 自取優惠：每斤折抵 $10
- **配送運費**：
  - 南區：滿 3 斤免運，未滿收 $100。
  - 其他台中地區：滿 5 斤免運，未滿收 $250。

## 👨‍💻 開發指引

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
# 若後端環境未建立，前端將自動切換為 Mock 模式
npm run dev
```

### 建立生產版本
```bash
npm run build
```
